📊 Nike Sales Data Analysis using Python

📌 Project Overview
This project focuses on performing Exploratory Data Analysis (EDA) on a sales dataset using Python. The analysis is conducted to extract meaningful insights from raw data and understand business performance across different dimensions such as revenue, profit, sales channels, and customer categories.

🎯 Objectives
Analyze sales data to identify trends and patterns

Understand relationships between key variables like Revenue, Profit, and Discount

Evaluate performance across Regions, Product Lines, and Sales Channels

Perform data visualization for better interpretation

Support data-driven decision-making

🛠️ Tools & Technologies
Python

Pandas

NumPy

Matplotlib

Seaborn

Visual Studio Code

📂 Dataset Description
The dataset includes the following features:

Product Name & Product Line

Sales Channel

Revenue & Profit

Discount & MRP

Units Sold

Gender Category

Region

Order Date

🔄 Project Workflow
1. Data Cleaning
Handled missing values

Converted data types (e.g., Order_Date to datetime)

Removed duplicates

2. Exploratory Data Analysis (EDA)
Used functions like groupby(), pivot_table(), corr(), and agg()

Performed statistical analysis using describe()

3. Data Visualization
Created multiple charts such as:

📊 Bar Charts

📈 Line Charts

🥧 Pie Charts

🔵 Scatter Plots

📉 Area Charts

4. Types of Analysis
Univariate Analysis → Single variable analysis

Bivariate Analysis → Relationship between two variables

Multivariate Analysis → Multiple variable interactions

Temporal Analysis → Time-based trends

📊 Key Insights
Revenue and Profit have a strong positive relationship

Sales Channels significantly influence overall performance

Discounts impact customer purchasing behavior

Certain Regions and Product Lines generate higher revenue

Sales show variation across different time periods

📌 Final Conclusion
The project demonstrates how data analysis techniques can transform raw data into actionable insights. Businesses can use these insights to optimize pricing strategies, improve sales channels, and better understand customer behavior.
